# 安全检查清单

> 版本：v1.0.0 | 维护于 `shared/standards/`

## 代码安全

- [ ] 无硬编码密钥、密码、Token
- [ ] 所有用户输入已验证和清洗
- [ ] SQL 使用参数化查询
- [ ] 无 XSS 风险（输出已转义）
- [ ] 无命令注入风险（禁止拼接 shell 命令）
- [ ] 无路径遍历风险（文件路径已规范化）

## 依赖安全

- [ ] 第三方依赖版本已锁定
- [ ] 无已知漏洞的依赖（通过安全扫描）
- [ ] 依赖来源可信

## 数据安全

- [ ] 敏感数据传输使用 HTTPS 加密
- [ ] 日志中不包含密钥和密码
- [ ] 用户数据处理合规

## 认证与授权

- [ ] API 接口有权限校验
- [ ] Token 有过期机制
- [ ] 关键操作有审计日志
