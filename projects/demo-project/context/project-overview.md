# Demo Project — 项目概述

> 本文件是 demo-project 的核心上下文文档，供 Skill 和 Agent 读取以理解项目全貌。

## 项目名称

Demo Project（智能订单管理系统）

## 项目简介

一个基于微服务架构的智能订单管理平台，支持订单创建、支付、物流追踪和数据分析。

## 团队

| 角色 | 成员 | 职责 |
|------|------|------|
| 架构师 | James | 系统设计、代码审查 |
| 后端开发 | Alice | 订单服务、支付服务 |
| 前端开发 | Bob | Web 控制台 |
| DevOps | Charlie | CI/CD、部署 |

## 核心业务流程

```
用户下单 → 库存校验 → 创建订单 → 支付处理 → 物流分配 → 状态通知
```
